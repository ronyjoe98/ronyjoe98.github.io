# Task 1: Personal Website

## Requirements Checklist
1. CSS selector for HTML tag: `body` tag styling  
2. Class used on 2+ elements: `.intro-text` class applied to multiple paragraphs  
3. ID on element: `#main-title` on the h1 element  
4. Two div sections: `.header-section` and `.content-section`  
5. Link to another website: Links to GitHub and LinkedIn  
6. Hover effect on links: Color change on `a:hover`  
7. Two box model properties: `padding` and `margin` on `.content-section` and `.portfolio-image`  
8. Image included: Placeholder image (replace with your own)  
9. Meta description tag: Included in `<head>` section  
10. GitHub Pages deployment: Follow instructions below

## Deployment Instructions

1. Create a new repository on GitHub named `yourgithubusername.github.io`
2. Clone the repository to your computer
3. Copy `index.html` and `stylesheet.css` into the cloned repository
4. Commit and push:
   ```bash
   git add .
   git commit -m "Initial commit - personal website"
   git push origin main
   ```
5. Your site will be available at `https://yourgithubusername.github.io`

## Customization

- Replace the placeholder image URL with your own image
- Update the content with your actual information
- Modify colors in `stylesheet.css` to match your preferences
- Add more sections as needed

## Note

Make sure to replace the placeholder image with a safe-for-work image before deploying!
